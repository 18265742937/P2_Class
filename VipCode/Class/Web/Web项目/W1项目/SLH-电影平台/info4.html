初代惊奇队长迈-威尔

迈-威尔（Mar-Vell），克里舰队尉官，属于克里人中肤色较浅的一族。他起先被至高智慧（Supreme Intelligence）派去地球做卧底，但迈-威尔对地球人产生了敬意，决定帮助他们，获得了惊奇队长的英雄称号。

至高智慧相信人类具有巨大的潜在精神力，为了夺取这种力量，他安排迈-威尔和里克·琼斯（Rick Jones）通过手环建立精神连接。当里克被困在负空间（Negative Zone）时，可以通过手环与迈-威尔换位，暂时回到地球；他们同样拥有了感知彼此思想的能力。克里星和斯克鲁星的战争中，至高智慧动用里克的精神力为自己效力，差点害死了他，迈-威尔用自己的生命力救了里克。他与斯克鲁族公主发生了恋情，留下一个儿子，即少年复仇者的小浩克（Hulking）

来自泰坦星的灭霸用宇宙立方自封上帝，迈-威尔联合复仇者击败了灭霸并摧毁宇宙立方，粉碎了灭霸的阴谋。在战斗生涯中，迈威尔曾遇到古老的宇宙灵体Eon，并获得能力“宇宙意识”，能感知宇宙中来自任何方位的危险。

迈-威尔意外进入负空间并无法脱身，但里克·琼斯找到了能量手环。每当地球有危险时，身处地球的里克·琼斯就与他进行位置切换，让迈-威尔暂时返回地球。一次与超能适体的战斗中，迈-威尔诱使超能适体戴上手环，使其进入负空间，与里克·琼斯成功从负空间脱身。

迈威尔不慎吸入毒气，患上了癌症。

《惊奇队长之死》中，迈-威尔在泰坦星度过了最后的时光。生命的最后时刻，他在超级英雄们的簇拥下陷入昏迷，在幻境中与已死的灭霸进行了最后一场战斗，与灭霸一起被死亡带走。超级英雄们将他安葬在泰坦星上。

迈-威尔与泰坦星的女性人工智能生命体伊莱亚斯（Elysius）有过一段恋情，留下后裔格尼斯-威尔（Genis-Vell）和菲拉-威尔（Phyla-Vell），两人都曾担任惊奇队长。
